<h1 id="migration-react" style={{fontFamily:"Nunito"}}> Migration depuis Material UI </h1>

<h2 id="introduction" style={{border:"none", fontFamily:"Nunito"}}> Introduction </h2>

<p> Ce guide compare l'architecture et les conventions de nommage de **Material UI (MUI)** avec celles du **Design System RTE**. Si vous migrez depuis Material UI, cette documentation vous aidera à comprendre les différences fondamentales entre les deux systèmes. </p>

<h2 id="architectural-differences" style={{border:"none", fontFamily:"Nunito"}}> Différences architecturales </h2>

<h3 id="component-model" style={{fontFamily:"Nunito"}}> Modèle de composants </h3>

<p> **Material UI** utilise des composants React avec des props pour la configuration. Le **Design System RTE** utilise également des composants React avec des props, mais avec une API unifiée et des conventions de nommage cohérentes. </p>

<h4 id="material-ui-approach" style={{fontFamily:"Nunito"}}> Approche Material UI </h4>

```tsx
import { Button, TextField, Checkbox } from '@mui/material';

<Button variant="contained" color="primary">Click me</Button>
<TextField label="Nom" variant="outlined" />
<Checkbox defaultChecked />
```

<h4 id="rte-design-system-approach" style={{fontFamily:"Nunito"}}> Approche Design System RTE </h4>

```tsx
import { Button, TextInput, Checkbox } from '@design-system-rte/react';

<Button variant="primary" label="Click me" />
<TextInput id="name" label="Nom" />
<Checkbox id="accept" label="Accepter" />
```

<h2 id="migration-examples" style={{border:"none", fontFamily:"Nunito"}}> Exemples de migration </h2>

<h3 id="button-migration" style={{fontFamily:"Nunito"}}> Migration d'un bouton </h3>

<h4 id="material-ui-button" style={{fontFamily:"Nunito"}}> Material UI </h4>

```tsx
import { Button } from '@mui/material';

<Button variant="contained" color="primary" disabled={isDisabled} onClick={handleClick}>
  Click me
</Button>
```

<h4 id="rte-design-system-button" style={{fontFamily:"Nunito"}}> Design System RTE </h4>

```tsx
import { Button } from '@design-system-rte/react';

<Button 
  variant="primary" 
  label="Click me"
  disabled={isDisabled} 
  onClick={handleClick}
/>
```

<h3 id="input-migration" style={{fontFamily:"Nunito"}}> Migration d'un champ de texte </h3>

<h4 id="material-ui-input" style={{fontFamily:"Nunito"}}> Material UI </h4>

```tsx
import { TextField } from '@mui/material';

<TextField 
  label="Nom" 
  variant="outlined"
  value={name}
  onChange={(e) => setName(e.target.value)}
  error={hasError}
  required
/>
```

<h4 id="rte-design-system-input" style={{fontFamily:"Nunito"}}> Design System RTE </h4>

```tsx
import { TextInput } from '@design-system-rte/react';

<TextInput
  id="name"
  label="Nom"
  value={name}
  onChange={setName}
  error={hasError}
  required
/>
```

<h3 id="checkbox-migration" style={{fontFamily:"Nunito"}}> Migration d'une case à cocher </h3>

<h4 id="material-ui-checkbox" style={{fontFamily:"Nunito"}}> Material UI </h4>

```tsx
import { Checkbox, FormControlLabel } from '@mui/material';

<FormControlLabel
  control={<Checkbox checked={isChecked} onChange={(e) => setIsChecked(e.target.checked)} />}
  label="Accepter les conditions"
  disabled={isDisabled}
/>
```

<h4 id="rte-design-system-checkbox" style={{fontFamily:"Nunito"}}> Design System RTE </h4>

```tsx
import { Checkbox } from '@design-system-rte/react';

<Checkbox
  id="accept"
  label="Accepter les conditions"
  checked={isChecked}
  onChange={(e) => setIsChecked(e.target.checked)}
  disabled={isDisabled}
/>
```

<h3 id="modal-migration" style={{fontFamily:"Nunito"}}> Migration d'un dialogue/modal </h3>

<h4 id="material-ui-modal" style={{fontFamily:"Nunito"}}> Material UI </h4>

```tsx
import { Dialog, DialogTitle, DialogContent, DialogActions, Button } from '@mui/material';

<Dialog open={isOpen} onClose={() => setIsOpen(false)}>
  <DialogTitle>Titre</DialogTitle>
  <DialogContent>
    Contenu du dialogue
  </DialogContent>
  <DialogActions>
    <Button onClick={() => setIsOpen(false)}>Annuler</Button>
    <Button onClick={handleValidate} variant="contained">Valider</Button>
  </DialogActions>
</Dialog>
```

<h4 id="rte-design-system-modal" style={{fontFamily:"Nunito"}}> Design System RTE </h4>

```tsx
import { Modal, Button } from '@design-system-rte/react';

<Modal
  id="my-modal"
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  title="Titre"
  description="Contenu du dialogue"
  primaryButton={<Button variant="primary" label="Valider" onClick={handleValidate} />}
  secondaryButton={<Button variant="neutral" label="Annuler" onClick={() => setIsOpen(false)} />}
/>
```

<h2 id="migration-checklist" style={{border:"none", fontFamily:"Nunito"}}> Checklist de migration </h2>

<ul>
  <li> Remplacer les imports Material UI par les imports du Design System RTE </li>
  <li> Adapter les props : <code>variant</code>, <code>size</code>, <code>label</code> </li>
  <li> Ajouter les <code>id</code> obligatoires pour l'accessibilité </li>
  <li> Mettre à jour la configuration de thème (attributs <code>data-theme</code> et <code>data-mode</code>) </li>
  <li> Tester l'accessibilité et les comportements interactifs </li>
</ul>
